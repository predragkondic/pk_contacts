# Variussystems Contacts

A simple contacts management tool.

## Requirements

* php >= 7.1
* php-intl
* php-mysql
* mysql db
* composer

## How to run

This project uses [composer](https://getcomposer.org/doc/00-intro.md) for project dependencies. 

`composer install`

Setup DB connection by copying .env.sample to .env and changing the credentials.
User the webserver of your choice.
